# Парсер фильмов с Википедии (Scrapy)

## Описание проекта

Данный репозиторий содержит Scrapy-парсер, который собирает информацию о фильмах
с русской Википедии, начиная со страницы:

https://ru.wikipedia.org/wiki/Категория:Фильмы_по_алфавиту

Парсер автоматически обходит страницы категории и подкатегорий, переходит
на страницы отдельных фильмов и извлекает структурированную информацию.

## Собираемые данные

Для каждого фильма собираются следующие поля:

- **Название**
- **Жанр**
- **Режиссёр**
- **Страна**
- **Год**

Если какое-либо поле отсутствует на странице фильма, в CSV-файле оно помечается
значением `не указано`.

⚠️ В результат сохраняются **только фильмы**  
(страницы про персонажей, вселенные, техники и т.п. отфильтровываются).

## Структура репозитория

├── film_spider.py # Scrapy-скрипт для парсинга фильмов

├── films.csv # Результат работы парсера (CSV)

└── README.md # Описание проекта

## Используемые технологии

- Python 3.8+
- Scrapy

## Установка и запуск

### 1. Клонирование репозитория


git clone https://github.com/<ваш_логин>/<имя_репозитория>.git
cd <имя_репозитория>
### 2. Установка зависимостей
Рекомендуется использовать виртуальное окружение:

`python -m venv venv`

`source venv/bin/activate`     # Linux / macOS

`venv\Scripts\activate`         # Windows

Установка Scrapy:

`pip install scrapy`

### 3. Запуск парсера
`python film_spider.py`

После завершения работы в папке проекта будет создан (или перезаписан) файл:
`films.csv`


CSV-файл сохраняется в кодировке UTF-8 и содержит следующие колонки:

`title, genre, director, country, year`

Пример строки:

38-я параллель (фильм),"драма, боевик, военный",Кан Джегю,Республика Корея,2004
